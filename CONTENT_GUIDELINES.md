# Content Guidelines for Auditik Website

Guidelines for creating and managing content using TinaCMS.

## 🎯 Blog Posts

### Creating a New Blog Post

1. Go to the website admin: **yoursite.com/admin**
2. Click **"Blog Posts"** in the left menu
3. Click **"+ New Post"** button

### Blog Post Fields

| Field | Description | Example |
|-------|-------------|---------|
| **Title** | Main article title | "5 Sinais de Perda Auditiva" |
| **Description** | SEO description (160 chars max) | "Conheça os principais sinais..." |
| **Author** | Who wrote it | "Auditik Equipe" or your name |
| **Publication Date** | Publish date | 2024-04-09 |
| **Category** | Article topic | Saúde Auditiva, Tecnologia, Dicas |
| **Featured Image** | Hero image for article | Upload JPG or PNG |
| **Featured Post** | Show on homepage (yes/no) | Check if important article |
| **Body** | Main article content | Write or paste Markdown |

### Writing Tips

✅ **DO:**
- Use clear, simple Portuguese
- Break text into short paragraphs
- Use bullet points for lists
- Include a call-to-action (CTA)
- Write for people, not search engines
- Proofread before publishing

❌ **DON'T:**
- Write very long paragraphs
- Use overly technical language
- Keyword stuff (repeat keywords)
- Copy content from competitors
- Publish without reviewing

### Article Topics to Consider

**Educational:**
- "Como escolher um aparelho auditivo"
- "Guia completo para limpeza do aparelho"
- "Tipos de perda auditiva"
- "Benefícios da detecção precoce"

**Product:**
- "Philips HearLink 100H: Tecnologia em seu ouvido"
- "Comparativa: HearLink vs. concorrentes"
- "Novidades em aparelhos auditivos 2024"

**Testimonials:**
- "História de Maria: 'Voltei a ouvir'"
- "Depoimento de cliente: Minha transformação"

**Promotions:**
- "Março com 15% de desconto"
- "Planos de financiamento para 2024"

## 🏥 Insurance Partners (Convênios)

### Managing Insurance Partners

1. Go to website admin: **yoursite.com/admin**
2. Click **"Insurance Partners"** in the left menu
3. Click **"+ New Partner"** or edit existing

### Partner Fields

| Field | Description | Example |
|-------|-------------|---------|
| **Partner Name** | Full name of the insurance/partnership | "Unimed Piracicaba" |
| **Description** | Brief overview | "Cobertura completa para aparelhos..." |
| **Benefits** | What customers get | "Cobertura 100%, Consultas grátis" |
| **Coverage** | Which devices are covered | "HearLink 100H, 500, 700" |
| **Contact Info** | How to verify coverage | "Central: 0800-123-4567" |
| **Logo** | Partner company logo | Upload PNG or SVG |

## 📝 Content Best Practices

### SEO Titles
- Keep under 60 characters
- Include main keyword
- Make it compelling

❌ Bad: "Blog"  
✅ Good: "5 Sinais de Perda Auditiva - Auditik"

### SEO Descriptions
- 150-160 characters
- Summary of content
- Include call-to-action

❌ Bad: "Artigo sobre audição"  
✅ Good: "Descubra os 5 sinais principais de perda auditiva e como detectar. Agende uma avaliação gratuita na Auditik."

### URLs (Auto-generated)
- No need to set manually
- Based on article title
- Use hyphens not underscores

Example: "5 Sinais de Perda Auditiva" → `/blog/5-sinais-de-perda-auditiva`

## 📸 Images

### Image Requirements

- **Format**: JPG or PNG
- **Size**: 1200x630px for blog featured images
- **File size**: Under 500KB (will be optimized)
- **Alt text**: Describe image content

### Image Locations

All images go into: **`/public/images/`**

TinaCMS will upload automatically. No need to manually add files.

### Image Tips

✅ Use high-quality images
✅ Show real patients (with permission)
✅ Use Auditik branding colors
✅ Keep consistent style
❌ Don't use watermarked images
❌ Don't use blurry photos

## 🔄 Publishing Workflow

### Workflow for Blog Posts

1. **Draft**: Write content without publishing
2. **Proofread**: Review for spelling/grammar
3. **Preview**: Check how it looks before publishing
4. **Publish**: Click "Save & Publish"
5. **Share**: Post on social media

### Republishing Changes

- Edit the post anytime
- Click "Save & Publish"  
- Website updates in ~5 minutes
- No need to inform developers

## 🚫 Content You Cannot Edit

- Footer information

## ✅ Pre-Publishing Checklist

Before clicking "Publish", verify:

- [ ] Title is clear and compelling
- [ ] SEO description written (150-160 chars)
- [ ] Featured image uploaded and optimized
- [ ] Content is spell-checked
- [ ] Links work correctly
- [ ] Category selected
- [ ] Author name correct
- [ ] Date is correct
- [ ] Call-to-action included (e.g., "Agende sua avaliação")
