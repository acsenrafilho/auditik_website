Check Point
The integration between Tuvis and Check Point leverages the firewall’s Application Control and URL Filtering capabilities to granularly block access to WhatsApp Web and similar messaging channels in corporate browsers. Through the Unified Security Policy, access is allowed exclusively via Tuvis Desktop, ensuring that network security policies extend to instant messaging without creating generic exceptions or introducing Shadow IT risks.
Key Features
- Block WhatsApp Web and other messaging channels using Check Point Application Control.
- Allow access exclusively through Tuvis Desktop within existing network security policies.
- Integrate with Check Point firewall policies without requiring parallel configurations.
Benefits for Your Business
- Network security without exceptions: Check Point blocks browser-based messaging channels and ensures access only through Tuvis Desktop.
- Unified security policy: Corporate messaging becomes part of the firewall’s control scope, eliminating security blind spots.
- Full traceability: All messaging channel usage is routed through Tuvis Desktop and logged for auditing purposes.
