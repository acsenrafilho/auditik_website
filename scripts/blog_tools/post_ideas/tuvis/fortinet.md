Fortinet
The integration between Tuvis and Fortinet FortiGate uses signatures and security profiles to manage access to messaging channels with granular control. Through firewall policies, access to WhatsApp via web browsers is blocked, while traffic is exclusively allowed through Tuvis. As a result, the existing network security and traffic inspection layers are extended to protect and audit corporate communication channels, eliminating network blind spots.
Key Features
- Blocking access to WhatsApp Web and other messaging channels through FortiGate URL filtering and application control policies.
- Routing messaging traffic to Tuvis Desktop as the only authorized access point.
- Applying existing FortiGate network security policies to corporate communication channels
Benefits for Your Company
- Extended network security for messaging: FortiGate policies also protect access to WhatsApp and Telegram channels.
- Traffic visibility: All messaging channel access is logged within FortiGate policies, providing complete traceability.
- Simplified management: The integration leverages the existing Fortinet infrastructure without requiring additional parallel solutions.
