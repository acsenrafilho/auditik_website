Broadcom
The integration between Tuvis and Broadcom security solutions leverages the corporate proxy policies of Symantec ProxySG (formerly Blue Coat) to manage and control web access. Through this configuration, access to WhatsApp Web and other messaging channels is blocked on standard browsers, allowing messaging traffic exclusively through the Tuvis platform. As a result, the Tuvis environment becomes the only authorized communication gateway, ensuring that every exchanged message passes through the platform’s Data Loss Prevention (DLP) and monitoring controls.
Key Features
- Granular blocking of WhatsApp and similar messaging channels via browser using filtering categories and security policies within the Symantec Web Gateway.
- Traffic redirection and authorization of messaging channels exclusively through Tuvis.
- Assurance that no parallel communication occurs outside the Tuvis-controlled environment.
Benefits for Your Company
- Protected access: employees cannot use messaging channels outside Tuvis, eliminating browser-based security gaps.
- Centralized control: existing ProxySG proxy policies are extended to cover corporate messaging channels.
- Guaranteed compliance: all communications are logged, traceable, and auditable since they occur exclusively within the managed Tuvis environment.
