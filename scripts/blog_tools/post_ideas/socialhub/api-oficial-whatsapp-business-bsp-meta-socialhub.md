API de WhatsApp: integre qualquer sistema ao WhatsApp
API REST com documentação completa para enviar e receber mensagens, gerenciar contatos, criar conversas e receber webhooks em tempo real. Integre seu ERP, CRM, e-commerce ou sistema próprio ao WhatsApp.
O que é a API de WhatsApp da SocialHub?
A API de WhatsApp da SocialHub é uma interface REST que permite integrar qualquer sistema ao WhatsApp de forma programática. Com ela, desenvolvedores enviam e recebem mensagens (texto, imagem, áudio, vídeo, PDF), criam e gerenciam contatos, aplicam tags, criam conversas, atribuem atendentes e recebem eventos em tempo real via webhooks. A documentação completa está disponível em ajuda.socialhub.pro com exemplos de requisição e respostas. A API se integra com todas as funcionalidades da plataforma: CRM, chatbot, atendimento e notificações. Disponível nos planos Plus (R$199/mês) e Master (R$399/mês).
O que você pode fazer com a API
Endpoints para mensagens, contatos e conversas
A API oferece endpoints completos para enviar mensagens de qualquer tipo, criar e gerenciar contatos com tags, abrir e fechar conversas, listar atendentes e consultar histórico. Tudo com autenticação por token Bearer e respostas em JSON.
- Envio de texto, imagem, áudio, vídeo e PDF
- CRUD completo de contatos e tags
- Criação e atribuição de conversas
- Consulta de histórico e status
Webhooks em tempo real
Receba notificações instantâneas quando eventos acontecem na plataforma: nova mensagem recebida, contato criado, conversa atribuída, mensagem lida e mais. Configure a URL do seu servidor e a SocialHub envia os eventos via POST.
- Evento: nova mensagem recebida
- Evento: mensagem enviada / entregue / lida
- Evento: contato criado ou atualizado
- Evento: conversa atribuída ou finalizada
Integração com CRM, chatbot e atendimento
A API não é isolada: ela se integra com todas as funcionalidades da SocialHub. Crie leads no CRM via API, acione fluxos de chatbot, atribua conversas a atendentes específicos e envie notificações programáticas — tudo no mesmo ecossistema.
- Crie leads no pipeline do CRM via API
- Acione fluxos de chatbot por evento
- Atribua conversas a atendentes específicos
- Integre com ERPs, e-commerces e sistemas próprios
