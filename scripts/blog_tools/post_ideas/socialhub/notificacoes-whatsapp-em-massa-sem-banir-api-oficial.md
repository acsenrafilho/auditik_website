Notificações e campanhas de WhatsApp: envio automático com rotação inteligente
Envie notificações de texto, áudio, vídeo e arquivos para sua base de contatos. Use a API Oficial ou conecte via QR Code, selecione múltiplos números e templates, e o sistema rotaciona automaticamente. Crie cadências pelo CRM com condicionais.
O que são notificações e campanhas de WhatsApp?
Notificações de WhatsApp são mensagens enviadas para sua base de contatos pelo WhatsApp, como promoções, avisos, cobranças, confirmações de pedido e códigos de rastreamento. A SocialHub oferece duas formas de envio: pela API Oficial do WhatsApp (com templates aprovados pela Meta) ou pela própria API da SocialHub via conexão por QR Code. O diferencial é a rotação inteligente: ao criar uma campanha, você sobe uma planilha de contatos, seleciona múltiplos números e templates, e o sistema distribui automaticamente entre eles. Além disso, é possível criar cadências pelo CRM — sequências automáticas de WhatsApp com condicionais (se leu, se respondeu, se não abriu). Planos a partir de R$99/mês.

Duas formas de envio: API Oficial e QR Code
Escolha a melhor forma para sua operação. A API Oficial do WhatsApp usa templates pré-aprovados pela Meta, ideal para notificações transacionais. Já a API da SocialHub via QR Code conecta seu número existente e permite envios mais flexíveis, sem necessidade de aprovação de templates.
- API Oficial: templates aprovados, alta entregabilidade
- API SocialHub (QR Code): flexível, sem aprovação de templates
- Use ambas simultaneamente na mesma conta
- Texto, imagens, áudios, vídeos e PDFs
Rotação inteligente de números e templates
Ao criar uma campanha, suba sua planilha de contatos, selecione múltiplos números de WhatsApp e vários templates de mensagem. O sistema rotaciona automaticamente entre os números conectados e os templates selecionados, distribuindo a carga e maximizando a taxa de entrega.
- Importe contatos via planilha (CSV/XLSX)
- Selecione 2, 3 ou mais números de WhatsApp
- Selecione múltiplos templates de mensagem
- Sistema rotaciona automaticamente entre eles
Cadência automática de WhatsApp pelo CRM
Suba uma base de contatos pelo CRM e coloque os leads em um fluxo de cadência automática. O fluxo inclui múltiplas mensagens de WhatsApp com intervalos programados e condicionais: se leu, se respondeu, se não abriu. Funciona tanto pela API Oficial quanto pela API da SocialHub via QR Code.
- Sequência de mensagens com intervalos (1d, 3d, 7d...)
- Condicionais: se leu, se respondeu, se não abriu
- Funciona com API Oficial e QR Code
- Leads entram na cadência automaticamente pelo CRM
