Template chat adalah pesan yang sudah disimpan dan bisa kamu gunakan dengan cepat di percakapan apa pun. Template ini sangat cocok untuk balasan umum, sambutan, atau FAQ. Kamu juga bisa menambahkan penanda dinamis untuk secara otomatis mempersonalisasi pesan dengan informasi dari lead, kontak, atau kartu perusahaan.
Apa itu template chat?
Template chat adalah pesan tersimpan yang dapat kamu gunakan di percakapan tanpa perlu mengetik ulang dari awal. Saat kamu mengetik kata yang sesuai dengan nama template, Kommo akan otomatis memberikan saran—jadi pastikan nama template singkat dan jelas agar mudah ditemukan.
Kamu bisa menambahkan penanda dinamis menggunakan tanda kurung siku [ ] untuk mempersonalisasi pesan. Misalnya, [Nama kontak] akan diganti menjadi nama lengkap kontak saat pesan dikirim.
Contoh penggunaan
Template chat di Kommo membantu tim kamu merespons lebih cepat, tetap konsisten, dan mempersonalisasi pesan secara otomatis. Kamu bisa:
- Mengirim balasan cepat untuk pertanyaan umum, salam, atau FAQ tanpa mengetik ulang.
- Memperoleh personalisasi pesan dengan penanda dinamis seperti [Nama kontak] atau [Nama perusahaan].
- Menjaga konsistensi komunikasi di seluruh tim, meskipun banyak agen menangani lead atau pelanggan yang sama.
Cara membuat template chat
Untuk membuat template:
-
Buka Pengaturan → Template. Di bagian Template umum, klik Tambahkan template baru.
-
Kamu juga bisa mengakses Template dari:
a. Klik tab Chat → Ikon roda gigi → Alat chat → Template.
b. Klik tab WhatsApp → Template.
-
Ketika kamu klik Tambahkan template baru, sebuah nama akan dibuat secara otomatis. Kamu bisa mengeditnya kapan saja dengan mengklik ikon pensil.
a. Tambahkan media: Lampirkan gambar atau file dengan mengklik Unggah atau menariknya ke dalam kotak.
b. Tulis pesanmu: Ketik di bagian Teks. Gunakan penanda dinamis dengan tombol [-] dan tambahkan emoji dengan ikon stiker.
c. Tambahkan tombol: Di bagian Tombol, sertakan tombol Balas cepat atau Call to action.
-
Klik Simpan saat selesai.
Untuk mengedit template:
-
Kembali ke Template umum, klik nama template, lakukan perubahan, lalu klik Simpan.
Untuk menghapus template:
-
Di Template umum, pilih kotak centang di samping template dan klik Hapus.
Cara menggunakan template chat
Kamu bisa menggunakan template chat dengan beberapa cara saat percakapan:
Ketik kata kunci
Mulailah mengetik sebuah kata dari nama templatemu di kolom chat. Kommo akan otomatis menyarankan template yang cocok.
Tips: Buat nama template tetap singkat dan unik agar cepat muncul dan tidak tercampur dengan yang lain.
Ikon petir
Klik ikon petir yang ada di kolom chat untuk melihat semua template. Pilih yang kamu inginkan, tinjau teksnya, lalu kirim.
Garis miring
Ketik / (garis miring) di kolom chat untuk membuka daftar template. Pilih template, cek teksnya, lalu kirim.
Beginilah tampilan template pesan setelah kamu memilihnya untuk digunakan dalam chat.
Setelah kamu memilih template, template itu akan muncul di kotak input dan tombol akan berubah menjadi Kirim template.
Jika kamu mengedit teksnya, tombol akan kembali menjadi Kirim, mengirimkan pesan biasa alih-alih template.
Kirim otomatis via Salesbot
Untuk mengirim template secara otomatis dengan Salesbot:
-
Buka editor visual Salesbot dan buat bot baru atau edit yang sudah ada.
-
Tambahkan langkah Pesan.
-
Klik Template dan pilih template-mu.
-
Simpan bot dan atur kondisi pemicu.
Kirim ke banyak lead via Siaran
Untuk menyiarkan template:
-
Buka jendela pengeditan Siaran.
-
Di bagian Konten, buat template baru atau pilih template yang sudah ada.
-
Atur penerima, channel, dan waktu peluncuran. Klik Kirim sekarang.
Pelajari lebih lanjut: Cara membuat siaran | Cara kustomisasi siaran.
