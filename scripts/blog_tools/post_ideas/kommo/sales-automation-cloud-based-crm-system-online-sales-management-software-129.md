Para configurar su primer activador automático de correo electrónico, necesitará:
- Kommo (plan avanzado o empresarial)
- Una dirección de correo electrónico que está conectada a Kommo
- Al menos 1 plantilla creada en la sección Correo
¿Tiene todo? Genial, comencemos.
Abra la sección Prospectos en la barra de menú izquierda.
- Para navegar a la Pipeline digital, haga clic en Configuración
- Encuentre la etapa en la que desea agregar el activador y haga clic en un espacio vacío
- Seleccione "Enviar un correo electrónico"
- Aparecerá una ventana:
Dentro de esta ventana, elija:
- Las condiciones para enviar el correo electrónico (por ejemplo, todos los prospectos con una etiqueta VIP)
- Cuándo ejecutar el activador
- La plantilla de correo electrónico para enviar
- El remitente (desde qué correo electrónico de usuario enviar el mensaje)
Nota: El correo electrónico se puede enviar desde dos tipos de remitentes:
- Una dirección de correo electrónico configurada como personal. El propietario de esta dirección de correo electrónico se establecerá como remitente.
- Una dirección de correo electrónico configurada como compartida. En este caso, debe seleccionar qué usuario mostrar como remitente.
- OPCIONAL: Marque la casilla "Agregar un enlace “cancelar suscripción” en el correo electrónico" para permitir que el destinatario cancele la suscripción de todos los correos electrónicos automáticos enviados desde Kommo.
- OPCIONAL: Marque la casilla "Aplicar el activador a todos los prospectos que ya están en la etapa" para enviar automáticamente el correo electrónico a los prospectos que ya están en la etapa del embudo.
¡Y eso es todo lo que hay para enviar correos electrónicos!
