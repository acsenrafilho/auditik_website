¿Alguna vez ha querido llegar a prospectos justo cuando visitan su sitio web? Con Kommo, puede configurar la automatización para que las acciones ocurran justo en el momento correcto
"Cuando se visita el sitio web seleccionado" es una opción de activación de automatización de la Pipeline Digital. Por ejemplo, puede crear una tarea cuando se visita el sitio o enviar al prospecto a una etapa diferente en su embudo.
Para configurar esta acción, diríjase a la Pipeline Digital haciendo clic en Prospectos, luego en "Configuración" en la sección superior derecha de la visualización de Prospectos. Dentro de esta visualización de configuración, puede crear cualquier activador y usar Ejecutar: cuando se visita el sitio web seleccionado.
Verá una ventana adicional abierta donde podrá configurar la URL del sitio, luego, si desea registrar la visita de todas las páginas. El código del script se enumerará aquí, el cual usted debe agregar al cuerpo de su sitio.
La función de código se llama Pixel y se utiliza para registrar el hecho de que un cliente visitó su sitio web. Una vez que Pixel registre la visita de este cliente exacto, ejecutará su activador de automatización, ¡así de simple! Si tiene configurado el "registro para todas las páginas", se ejecutará en cada rincón de su sitio.
