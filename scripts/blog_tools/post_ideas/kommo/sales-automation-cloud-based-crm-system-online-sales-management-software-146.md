Preenchendo dados de campos customizados
O Salesbot pode alterar os campos personalizados. É uma ótima maneira de solicitar um endereço de e-mail ou número de telefone no chat. Para escolher o campo que você deseja solicitar ao usuário do bate-papo, selecione Definir campo.
Escolha qual campo você deseja preencher e que tipo de dados você precisa.
Você pode fazer com que o Salesbot preencha os seguintes campos:
- Lead: ID - Receba o número de identificação do lead
- Lead: Origem - Receba a origem do lead
- Lead: ID do usuário - Receba o ID do usuário
- Contato: ID - Receba o ID do contato
- Mensagem do cliente - Receba a última mensagem de bate-papo que o usuário enviou antes que esta ação fosse engatilhada.
- Entrada manual - o Salesbot entrará em contato com o usuário responsável no Kommo e solicitará que ele insira os dados manualmente
