Para criar um usuário gratuito, vá para a seção "Configurações" - "Usuários". Clique em "Adicionar usuário", selecione "Grupo" - "Usuários gratuitos".
Una vez haz hecho esto, el usuario recibirá sólo uno de estos dos tipos de correos electrónicos:
1) Si el usuario nunca ha utilizado Kommo, recibirá un correo electrónico donde encontrará:
- El nombre del usuario que lo invitó;
- Datos de autorización;
- Información sobre Kommo.
2) Si el usuario ya está registrado en la base de datos de Kommo (hay/hubo otra cuenta de Kommo), recibirá un correo electrónico que contiene:
- El nombre del usuario que lo invitó;
- El nombre de usuario y la contraseña sin modificaciones;
- Información sobre Kommo.
Para dar acceso a un usuario gratis a la tarjeta de leads o de clientes, es necesario enviarle un chat interno para concederle acceso.
El usuario gratis verá una notificación en el centro de notificaciones y podrá ir a la tarjeta de leads/clientes.
