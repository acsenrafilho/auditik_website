Pengisian Data Kolom Kustom
Salesbot dapat mengubah kolom kustom. Salesbot memudahkan untuk memintakan alamat email dan nomor telepon di dalam percakapan. Jika ingin memilih bagian yang ingin kamu mintakan user untuk isi, pilih bagian Atur kolom.
Pilih kolom mana yang ingin diisi dan data apa yang kamu butuhkan.
Kamu dapat meminta Salesbot mengisi beberapa bagian berikut:
- Lead: ID - Menerima nomor ID dari lead.
- Lead: Asal - Menerima Asal dari lead.
- Lead: User ID - Menerima ID user.
- Kontak: ID - Menerima ID kontak.
- Client message - Menerima pesan terakhir yang user kirimkan sebelum tindakan ini ter-trigger.
- Manual Input - Salesbot akan menghubungi pengguna Kommo dan menanyakan mereka untuk memasukkan data secara manual.
