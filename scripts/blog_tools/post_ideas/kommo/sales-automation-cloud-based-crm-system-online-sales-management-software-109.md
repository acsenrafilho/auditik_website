Los ajustes preestablecidos personalizados le permiten mapear automáticamente todos los campos, de esta manera no tendrá que seleccionar campos manualmente cada vez que desee crear un nuevo prospecto. Esto es especialmente útil si utiliza con frecuencia un sitio web específico para buscar nuevos datos de contacto de prospectos.
Las instrucciones sobre cómo crear un ajuste preestablecido se pueden encontrar en la extensión.
Crear un ajuste preestablecido le permitirá obtener información de perfil de LinkedIn y completar los campos de la tarjeta de prospectos de Kommo en función de sus necesidades comerciales específicas.
