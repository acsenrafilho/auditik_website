A localização de widgets pode ser complicada, especialmente se você não estiver familiarizado com a terminologia em outros idiomas.
Para ajudar, compilamos uma lista dos termos mais comumente usados no sistema Kommo. Nós **recomendamos fortemente** que você use as traduções corretas, pois o uso de sinônimos pode confundir seus clientes.

| Idioma | Palavra |
| --- | --- |

| Inglês | Company |
| Espanhol | Compañía |
| Turco | Şirket |
| Indonésio | Perusahaan |
