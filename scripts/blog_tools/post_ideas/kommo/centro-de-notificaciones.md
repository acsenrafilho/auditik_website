title: "Centro de notificaciones"

---

El centro de notificaciones es una nueva funcionalidad del sistema de Kommo, que proporciona una entrega conveniente y confiable de alertas de la integración al usuario final.
La API de Kommo permite a los desarrolladores enviar notificaciones en un solo formato a un solo centro de notificaciones, informando rápidamente al usuario sobre el estado del trabajo de integración.
Existen varios canales de entrega para las notificaciones.
Este canal de entrega no puede ser deshabilitado por el usuario. Todos los nuevos mensajes se mostrarán en la interfaz de Kommo.
Este canal envía notificaciones por correo electrónico. El usuario puede elegir enviar el mensaje en el momento del evento o una vez al día.
Cuando ocurre un evento, el usuario recibirá una notificación push en la aplicación móvil. Para que este canal de entrega de mensajes funcione correctamente, necesitas configurar los permisos para la aplicación móvil y permitir que los mensajes emergentes se muestren.
Si el canal de notificación está activo, el sistema enviará notificaciones push al navegador y se permitirán las notificaciones en el navegador.
En el perfil del usuario, están disponibles ajustes que te permiten cambiar el orden en el que se reciben las notificaciones y, para cada proveedor, elegir los canales de entrega de los mensajes.
Puedes habilitar o deshabilitar todas las notificaciones moviendo el radio button Recibir notificaciones.
Los ajustes muestran todos los proveedores de notificaciones disponibles.
Para configurar los canales de entrega, debes seleccionar los que necesitas, configurar el modo de recepción de notificaciones si el canal Correo se encuentra activo y hacer clic en el botón Guardar.
