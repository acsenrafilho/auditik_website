- Select Typeform in Settings > Integrations
- Click on Install and then Sign in
- Authorize the connection
- Select the form you want to sync
- Map the fields where answers will be saved
- Click on Save
- In the pipeline, click on Setup pipeline > Add source > Typeform
That's it, you are all set up. 😎
When the synced form is filled out, a new lead will appear in the Incoming leads stage.
