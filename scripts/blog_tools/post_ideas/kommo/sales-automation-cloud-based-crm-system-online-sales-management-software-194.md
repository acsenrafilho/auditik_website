Como administrador de uma conta Kommo, você tem a opção de desativar ou remover usuários da conta. Um usuário desativado será marcado como inativo e não poderá usar o sistema, enquanto um usuário removido será removido completamente da conta.
Neste artigo, você aprenderá:
Como desativar um usuário
Vá para Configurações > Usuários, clique na caixa de seleção ao lado do usuário que você deseja desativar.
Selecione a ação desativar.
O sistema permitirá que você atribua os dados do usuário a qualquer um dos outros usuários ativos, se desejar.
Depois que o usuário for desativado, eles serão movidos para a guia inativa da seção de usuários.
Para acessar essa guia, clique na função Pesquisar e filtrar.
Lá, você pode ativar o usuário com facilidade novamente, clicando na caixa de seleção e selecionando ativar.
Removendo um usuário
Na caixa de seleção do usuário, selecione remover usuários.
O sistema permitirá reatribuir todos os dados do usuário para ouro usuário. O usuário selecionado será completamente removido da conta.
