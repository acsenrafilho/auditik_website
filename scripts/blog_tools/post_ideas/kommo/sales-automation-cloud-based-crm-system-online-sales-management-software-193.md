Para criar um usuário gratuito, vá para a seção "Configurações" - "Usuários". Clique em "Adicionar usuário", selecione "Grupo" - "Usuários gratuitos".
Feito isso, o usuário receberá apenas um desses dois tipos de e-mails:
- Caso o usuário nunca tenha usado a Kommo, ele receberá um e-mail onde encontrará
O nome do usuário que o convidou;
- Dados de autorização;
- Informações sobre Kommo.
- Caso o usuário já esteja cadastrado no banco de dados Kommo (existe/havia outra conta Kommo), ele receberá um e-mail que contém:
- O nome do usuário que o convidou;
- O nome de usuário e senha sem modificações;
- Informações sobre Kommo.
Para dar a um usuário acesso gratuito aos leads ou ao cartão do cliente, você precisa enviar-lhe um bate-papo interno para conceder acesso.
O usuário gratuito verá uma notificação na central de notificações e poderá acessar o cartão de leads/clientes.
