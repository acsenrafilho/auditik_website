Puedes crear etiquetas de dos formas diferentes
- En la interfaz del lead
- De forma masiva para 2 o más leads al mismo tiempo
Para crear una etiqueta en una lead, sólo tienes que abrirla y en la sección superior, justo debajo del nombre del lead (y cerca del ID del lead) encontrarás el botón #AGREGAR ETIQUETAS. Haz clic en él y comienza a escribir el nombre de tu nueva etiqueta, luego haz clic en la opción que aparece y tu etiqueta se creará, no es necesario hacer clic en guardar ya que se guarda automáticamente. para eliminarla sólo tienes que hacer clic en ella un par de veces.
Nota: Si eliminas una etiqueta de todos los leads, esta no se eliminará de la cuenta. Este artículo también describe cómo eliminar las etiquetas completamente.
Para añadir etiquetas en forma masiva, selecciona varios elementos en la vista de lista y haz clic en editar etiquetas
Verás una ventana emergente en la que puedes seleccionar una etiqueta para todas estas los leads seleccionados o crear una nueva con sólo escribirla en el campo de búsqueda, elige las etiquetas que necesites y haz clic en 'Guardar'.
