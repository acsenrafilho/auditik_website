Completar los datos de los campos personalizados
El Salesbot puede cambiar los campos personalizados. Es una gran manera de solicitar una dirección de correo electrónico o un número de teléfono en el chat. Para elegir el campo que desea solicitar al usuario del chat, seleccione Fijar campo.
Elija qué campo quiere llenar y qué tipo de datos necesita.
Puede hacer que el Salesbot complete los siguientes campos:
- Lead: ID - reciba el número de identificación del lead
- Lead: Origen - reciba el origen del Lead
- Lead: ID de usuario - reciba la ID de usuario
- Contacto: ID - Reciba la ID del contacto
- Mensaje del cliente: reciba el último mensaje de chat que envió el usuario antes de que se activará esta acción.
- Entrada manual: el Salesbot se comunicará con el usuario responsable en Kommo y le pedirá que ingrese los datos manualmente
