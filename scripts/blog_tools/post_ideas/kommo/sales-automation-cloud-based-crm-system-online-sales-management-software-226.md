If you already have a Kommo account and want to create one more, you can do that in a few clicks.
Note: If you want a new account for a different work process, you can consider creating a different pipeline instead.
-
Start by opening Kommo’s homepage and clicking on your profile name in the upper right corner.
Note: If you are not signed in, you need to do so first.
-
Go to Create new account
-
Enter new account name and account subdomain and finish by clicking on Create account.
And there you go! You can now set up your new account as you prefer.
