Salesbot
Click the ‘+’ button located under a pipeline stage you want to place it and select Salesbot from the list of triggers. -
Add a condition - you can specify the criteria for which leads the Salesbot should work. For instance, you can set the Salesbot to run for leads that have a specific responsible user or a particular tag. You can also add multiple conditions to ensure that the Salesbot only operates on leads that meet your desired criteria. -
Set the execute rules - with this option, you can choose from the list of triggers that launch the Salesbot. For example, you can choose the bot to start working when ‘Immediately a new bot is created on this stage’. To learn more about these triggers, visit our Salesbot triggers article. -
Set the activity time - use this option to set the activity time for your bot. For example, you can choose your bot to be on 24/7 or, only when you are away from the office. -
Leave messages unanswered - by switching on this toggle messages answered by the bot will be marked as unanswered. -
Choose a salesbot from the drop-down menu or create a new one right away. To learn more about how to create a salebot, visit our ‘ How to create a Salesbot ’ article. -
If you want this trigger to apply to all the leads already in the stage where you placed it, check the box and click Save .
Add task
Click the ‘+’ button under the pipeline stage you want to place it and select Add Task from the list of triggers. -
Add a condition - add one or multiple conditions that need to be met in order for the trigger to start working. For example, you can choose VIP tag as a condition and the task will be created only for the leads on this stage with VIP tag. -
Set the execute rules - choose from the list of triggers that activate the action. For example, you can choose to create a task when a new lead is created.To learn more about these triggers, visit our Salesbot triggers article. -
Set task deadline - set a deadline for your tasks that are created via this trigger. You can choose the following options: Immediately End of the day In 1 day In 3 days End of the week In days, hours, minutes
-
Choose the responsible user . If you choose the current responsible user, the task is assigned to the responsible user of the lead. You can also select any of your users from the list. -
Define the task type. By default, you have Follow up and Meeting. You can create more task types by going to Calendar > three dots button > Manage task types. -
Add a comment if necessary. It will be displayed in your task. -
If you want this trigger to apply to all the leads already in the stage where you placed it,, check the box and click Save .
Create lead
Click the ‘+’ button under a pipeline stage that you want to place it and select Create Lead from the list of triggers. -
Add a condition - add one or multiple conditions that need to be met in order for the trigger to start working. This means that you can set specific requirements, such as leads with specific tags, fields and other requirements. -
Set the execute rules - set rules for when this trigger should be launched. These rules include different kinds of triggers, such as pipeline triggers, scheduled triggers, behavior-based triggers and conversational triggers. To learn more about these triggers, visit our Salesbot triggers article. -
Create lead with - check the fields that you want to be transferred to a new lead. The new lead gets the info that you selected from the old lead. sales value tags contact company other
-
Choose the responsible user - keep the responsible user of the lead it is being copied from or just assign a new user from the list of your users. -
Specify the Pipeline and pipeline stage that the new leads should be created in. -
If you want this trigger to apply to all the leads already in the stage where you placed it,, check the box and click Save .
Send email
-
Click the ‘+’ button under a pipeline stage that you want to place it and select Send Email from the list of triggers. -
Add a condition - conditions are parameters that must be fulfilled in order to activate the trigger. -
Set the execute rules - this option determines when the trigger will start working. For example, if you select the "When moved to this stage" option, the trigger will be activated for all leads that reach this stage of the pipeline. -
Select the email template - choose a template you have created, and you can even add or modify them by going to "go to template settings" at the bottom of the menu.
-
Choose the sender - Choose who should be the sender of the email - it can be either the responsible user of the lead, corporate mailbox, or a specific user. If the user has not connected an email, it will be sent from the connected team's shared corporate email address. -
Finally, you can also configure these conditions : -
Apply only to the main contact - only main email contact of the lead is contacted -
Add an "unsubscribe" link in the email - your message will have an unsubscribe link, which will unsubscribe the user from automatic emails. -
Apply to all leads in this stage - The trigger will be applied to all leads in the stage if they fall under the rules.
-
Send webhook
Click the ‘+’ button under a pipeline stage that you want to place it and select Send Webhook from the list of triggers. -
Add a condition - add conditions to narrow down the leads that this trigger should work for. -
Choose the execute rule - choose the conditions to define when this trigger should start working out. To learn more about these triggers, visit our Salesbot triggers article. Add an URL to transfer data.
Change lead stage
-
Click the ‘+’ button under a pipeline stage that you want to place it and select Change Lead Stage from the list of triggers. -
Add a condition to clarify what leads this trigger should work for. -
Add execute rules - define when this trigger should start working by choosing the conditions. To learn more about these triggers, visit our Salesbot triggers article. -
Choose a pipeline and pipeline stage where the lead should be moved to. Keep in mind that it does not necessarily have to be the next stage, you can move them to any stage of the pipeline and even between different pipelines. Finally, click done when you finish setting up.
Add tags
-
Click the ‘+’ button under a pipeline stage that you want to place it and select Edit Tags stage from the list of triggers. -
Add a condition - choose the parameters that should be included in a lead in order to trigger start working. -
Add execute rules - set triggers to define when this trigger should be activated. To learn more about these triggers, visit our Salesbot triggers article. -
Choose the action - decide if you want to Add tag or Remove tag. -
Choose the tag (s) that you want to add or remove -
If you want this trigger to apply to all the leads already in the stage where you placed it, check the box and click Save .
Complete tasks
-
Click the ‘+’ button under a pipeline stage that you want to place it and select Complete Tasks from the list of triggers. -
Add a condition - you can include specific conditions that must be met to activate the trigger. -
Add execute rules - this determines when the trigger will be activated according to your configuration. To learn more about these triggers, visit our Salesbot triggers article. -
Choose the task type - choose the task type that needs to be completed automatically. If you want the trigger to apply to all kinds of tasks, just go with All. -
Set the task deadline - it is also necessary that when setting this trigger you specify the deadline of the task. These can be: All, Overdue, Due today or Due tomorrow. -
Add a comment - the comment you add will appear after the task is completed. -
If you want this trigger to apply to all the leads already in the stage where you placed it, check the box and click Save .
Generate form
-
Click the ‘+’ button under a pipeline stage that you want to place it and select Generate Form from the list of triggers. -
Add a condition - the parameters that must be met for the trigger to be activated. -
Add execute rules - under which actions the trigger will be activated. To learn more about these triggers, visit our Salesbot triggers article. -
Choose the type of form - you can choose between two forms: -
Individual form - update existing lead card information. This trigger creates a link to a form that you can send to your customers to fill out to automatically update the information inside the lead card. -
Reference form - create a new lead card. You also have the possibility to make a referral form, which creates a link with a field to identify the referrer instead of updating the current one. This is an excellent option for referral programs.
-
-
Create form - choose one of the templates and customize it to your needs.
-
Add tags to differentiate the leads that are received from your form. -
Choose the change stage - you can choose to move the leads that submitted the form to a different stage or pipeline. -
If you want this trigger to apply to all the leads already in the stage where you placed it, check the box and click Save .

Change field
-
Click the ‘+’ button under a pipeline stage that you want to place it and select Change field from the list of triggers. -
Add a condition - the conditions are the parameters that will activate the trigger. -
Add execute rules - this is the action that will activate the trigger. For example, when an email is received. To learn more about these triggers, visit our Salesbot triggers article. -
Choose the field In this section choose the pipeline stage that you want to edit. After you choose a field, you need to write what the field should include. For example, I am entering the zoom link that needs to be entered when moved to this stage. -
If you want this trigger to apply to all the leads already in the stage where you placed it, check the box and click Save .
Delete files
-
Click the ‘+’ button under a pipeline stage that you want to place it and select Delete Files from the list of triggers. -
Add a condition - determine the specific actions that must be fulfilled for a trigger to be activated. -
Add execute rules - determine when the trigger will be activated. To learn more about these triggers, visit our Salesbot triggers article. -
Select what files you want to delete. In Delete files option you can choose to delete: Select all, Used in fields, Attached in notes, Uploaded in media tab or Sent in chats -
Select the type file to be deleted. It can be svg or mp4. -
Set the size range to define what files should be deleted. This way, you can delete the big files to make more space in your cloud. -
If you want this trigger to apply to all the leads already in the stage where you placed it, check the box and click Save .
