Os modelos de chat são mensagens pré-salvas que você pode usar rapidamente em qualquer conversa. São perfeitos para respostas comuns, saudações ou perguntas frequentes. Você pode adicionar marcadores dinâmicos para personalizar automaticamente as mensagens com informações de um lead, contato ou cartão da empresa.
O que é um modelo de chat?
Um modelo de chat é uma mensagem salva que você pode usar em conversas sem precisar digitá-la do zero. Ao digitar uma palavra que corresponda ao nome de um modelo, a Kommo vai sugerir ele automaticamente — portanto, mantenha os nomes dos modelos curtos e distintos para encontrá-los rapidamente.
Você pode adicionar marcadores dinâmicos usando colchetes [ ] para personalizar as mensagens. Por exemplo, [Nome do contato] será substituído pelo nome completo do contato quando a mensagem for enviada.
Casos de uso
Os modelos de chat da Kommo ajudam sua equipe a responder mais rápido, manter a consistência e personalizar mensagens automaticamente. Você pode:
- Enviar respostas rápidas a perguntas comuns, saudações ou perguntas frequentes sem precisar digitar novamente.
- Personalizar mensagens usando marcadores dinâmicos como [Nome do contato] ou [Nome da empresa].
- Manter a comunicação consistente em toda a equipe, mesmo quando vários agentes atendem o mesmo lead ou cliente.
Como usar modelos de chat
Você pode usar modelos de chat de várias maneiras durante uma conversa:
Digitação de palavras-chave
Comece a digitar uma palavra do nome do modelo. A Kommo vai sugerir modelos correspondentes automaticamente.
Dica: Mantenha os nomes dos modelos curtos e únicos para que apareçam rapidamente e não se misturem com outros.
Ícone de raio
Clique no ícone de raio para ver todos os modelos. Selecione o que você deseja, revise o texto e envie.
Comando de barra
Digite / (barra) para abrir a lista completa de modelos. Escolha seu modelo, verifique o texto e envie.
Veja como o modelo de mensagem aparecerá quando você selecioná-lo para usar em um chat.
Após selecionar um modelo, ele aparecerá na caixa de entrada e o botão mudará para Enviar modelo.
Se você editar o texto, o botão retornará para Enviar, enviando uma mensagem normal em vez do modelo.
Enviar automaticamente via Salesbot
Para enviar um modelo automaticamente com um Salesbot:
-
Abra o editor visual do Salesbot e crie um novo bot ou edite um existente.
-
Adicione uma etapa de Mensagem.
-
Clique em Modelo e escolha seu modelo.
-
Salve o bot e defina uma condição de gatilho.
Envie para vários leads via transmissão
Para transmitir um modelo:
-
Abra a janela de edição de Transmissão.
-
Em Conteúdo, crie um novo modelo ou selecione um existente.
-
Defina os destinatários, o canal e o horário de início. Clique em Enviar agora.
