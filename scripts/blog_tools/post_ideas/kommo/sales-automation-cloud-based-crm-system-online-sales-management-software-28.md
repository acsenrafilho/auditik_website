Chat templates are pre-saved messages you can quickly use in any conversation. They’re perfect for common replies, greetings, or FAQs. You can add dynamic markers to automatically personalize messages with info from a lead, contact, or company card.
What is a chat template?
A chat template is a saved message you can use in conversations without typing it from scratch. When you type a word that matches a template’s name, Kommo will suggest it automatically—so keep template names short and distinct to find them quickly.
You can add dynamic markers using square brackets [ ] to personalize messages. For example, [Contact name] will be replaced with the contact’s full name when the message is sent.
Use cases
Chat templates in Kommo help your team reply faster, stay consistent, and personalize messages automatically. You can:
- Send quick responses to common questions, greetings, or FAQs without retyping.
- Personalize messages using dynamic markers like [Contact name] or [Company name].
- Keep communication consistent across the team, even when multiple agents handle the same lead or customer.
How to use chat templates
You can use chat templates in several ways during a conversation:
Keyword typing
Start typing a word from the template’s name. Kommo will suggest matching templates automatically.
Tip: Keep template names short and unique so they appear quickly and don’t get mixed up with others.
Lightning icon
Click the lightning icon to see all templates. Select the one you want, review the text, and send.
Slash command
Type / (slash) to open the full template list. Pick your template, check the text, and send.
Here’s how the message template will appear once you select it to use in a chat.
Once you select a template, it appears in the input box and the button changes to Send template.
If you edit the text, the button switches back to Send, sending a regular message instead of the template.
Automatically send via Salesbot
To send a template automatically with a Salesbot:
-
Open the Salesbot visual editor and create a new bot or edit an existing one.
-
Add a Message step.
- Click Template and choose your template.
-
Save the bot and set a trigger condition.
Send to multiple leads via Broadcasting
To broadcast a template:
-
Open the Broadcast editing modal.
-
In Content, create a new template or select an existing one.
-
Set recipients, channel, and launch time. Click Send now.
