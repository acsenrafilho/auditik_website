A central de notificações é um novo recurso do sistema Kommo, que oferece entrega conveniente e confiável de alertas da integração para o usuário final.
A API d Kommo permite que os desenvolvedores enviem notificações em um único formato para uma única central de notificações, informando prontamente o usuário sobre o status do funcionamento da integração.
Existem vários canais de entrega para notificações.
Esse canal de entrega não pode ser desativado pelo usuário. Todas as novas mensagens serão exibidas na interface da Kommo.
Esse canal envia notificações por e-mail. O usuário pode optar por receber a mensagem no momento do evento ou uma vez por dia.
Quando um evento ocorre, o usuário receberá uma notificação push no aplicativo móvel. Para que este canal de entrega de mensagens funcione corretamente, é necessário configurar as permissões para o aplicativo móvel e permitir a exibição de mensagens pop-up.
Se o canal de entrega estiver ativo, o sistema enviará uma notificação push para o navegador, desde que as notificações estejam permitidas no navegador.
No perfil do usuário, estão disponíveis configurações que permitem alterar a ordem de recebimento das notificações e, para cada provedor, escolher os canais de entrega de mensagens.
É possível ativar ou desativar todas as notificações movendo o botão de alternância Receber notificações.
As configurações exibem todos os provedores de notificação disponíveis.
Para configurar os canais de entrega, selecione os desejados, defina o modo de recebimento de notificações se o canal Email estiver ativo e clique no botão Salvar.
