Pernahkah kamu ingin menghubungi calon klien tepat setelah mereka mengunjungi situs web kamu? Dengan Kommo, kamu bisa menyetel automasi actions tertentu agar ia aktif di momen yang tepat.
“When selected website is visited” adalah automasi trigger dalam Digital Pipeline. Contohnya, kamu bisa membuat sebuah task saat situs sedang dikunjungi atau mengirimkan lead ke tahapan lain di pipeline kamu.
Untuk menyetel action ini, pergi ke Digital Pipeline dengan mengklik Lead, lalu klik “Pengaturan” di bagian kanan atas tampilan Lead. Di dalam tampilan penyetelan ini, kamu bisa membuat trigger apapun dan menggunakan Eksekusi: saat situs web terpilih dikunjungi.
Kamu akan melihat sebuah tambahan jendela yang terbuka dimana kamu bisa menyetel url situs, lalu jika kamu ingin merekam kunjungan semua laman, kode Script akan ditampilkan di sini yang nantinya kamu harus tambahkan ke body dari situs kamu.
Fungsi kode disebut sebuah Pixel, dan ini digunakan untuk mendaftarkan fakta bahwa klien sudah mengunjungi situs web kamu. Setelah Pixel mendaftarkan sejarah kunjungan klien kamu ini, automasi trigger. Jika kamu menyetel “record for all pages”, penyetelan ini akan berjalan di setiap sudut situs kamu.
